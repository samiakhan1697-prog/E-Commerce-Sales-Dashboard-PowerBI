
# 📊 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview
The **E-Commerce Sales Dashboard** is an interactive Business Intelligence project developed using Power BI.  
This dashboard provides valuable insights into sales performance, product categories, and overall business profitability.

The objective of this project is to help businesses analyze sales data efficiently and make data-driven decisions through clear and interactive visualizations.

---

## 🎯 Project Objectives
- Analyze overall **sales performance**
- Identify **top performing product categories**
- Track **sales trends over time**
- Evaluate **profitability and business growth**
- Provide an **interactive dashboard for better decision making**

---

## 🛠 Tools & Technologies Used
- **Power BI** – Data visualization and dashboard creation  
- **Data Analysis Techniques**  
- **Business Intelligence Concepts**

---

## 📊 Dashboard Features
The dashboard includes the following key components:

✔ **KPI Cards**
- Total Sales  
- Total Profit  
- Total Quantity Sold  

✔ **Sales Trend Analysis**
- Visual representation of sales over time.

✔ **Category-wise Performance**
- Analysis of product categories and their contribution to total sales.

✔ **Interactive Filters / Slicers**
- Allows users to filter and explore data dynamically.

✔ **Data Visualization**
- Charts and graphs that simplify complex business data.
