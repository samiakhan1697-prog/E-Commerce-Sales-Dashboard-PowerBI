# E-Commerce Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard developed to analyze e-commerce sales performance. The dashboard helps businesses understand sales trends, product category performance, and profit insights.

## Tools Used
- Microsoft Power BI
- Data Visualization
- Business Intelligence Techniques

## Dashboard Features
• KPI cards showing Total Sales, Profit and Quantity  
• Category wise sales analysis  
• Sales trend over time  
• Interactive slicers for filtering data  
• Visual comparison of product performance
